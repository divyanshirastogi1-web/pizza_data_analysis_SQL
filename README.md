# Pizza Hut Sales Analysis

# Project Overview

The Pizza Hut Sales Analysis Dashboard is an end-to-end Business Intelligence project that analyzes pizza sales using MySQL Workbench and visualizes key performance metrics through an interactive Microsoft Excel Dashboard.

The goal is to convert raw transactional data into meaningful insights that support better business decisions.

# Objectives

✅ Analyze overall sales performance

✅ Calculate business KPIs

✅ Discover customer purchasing behavior

✅ Identify top and bottom-selling pizzas

✅ Study monthly and daily sales trends

✅ Build an interactive dashboard using Excel
